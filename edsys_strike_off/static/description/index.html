Student Strike off

Student strike of it the deactivation of a student profile owing to absence of student from the school premises for a time period surpassing the regulations.
Student strike off feature can be initiated in case of low attendance, or long unexplained absence from school.

Once student is striked-off, he/she no longer appears in the student module. This keep data organized.

It is possible to re-activate the stagnant profile and get the student back to the dynamic student module.

Features

  • Deactivation /re-activation of profile
  • Strike of history
  • Overview of deactivated profile
